SO, What is a limit?

A limit is the value a function approaches as the variable of the function gets closer & closer to a particular value.

Limit Notation:

Note:
Notice how the word "approaches" is used.
This means that f(c) does not have to equal L.
It just has to approach L from both sides.

We write if the value \(f(x)\) approaches L as x approaches c.

One-Sided & Two-Sided Limits

\(\lim_{{x \to 2}} f(x)\) is the number that \(f(x)\) approaches as x approaches 2 from both sides.

here, \(\lim_{{x \to 2}} f(x) = L_1\)
This is a two-sided limit (aka the general limit)

\(\lim_{{x \to 2^+}} f(x)\) is the number that \(f(x)\) approaches as x approaches 2 only with values greater than 2.

\(\rightarrow\) this is called a right-hand limit.

\(\lim_{{x \to 2^-}} f(x)\) is the number that \(f(x)\) approaches as x approaches 2 only with values less than 2.

\(\rightarrow\) this is called a left-hand limit.

\(\lim_{{x \to 2^+}} f(x) = L_2\)
\(\lim_{{x \to 2^-}} f(x) = L_3\)

Algebraic Properties of Limits

Assuming that the limits exist,
  1. if \(b\) is a constant, then \(\lim_{{x \to c}}(bf(x)) = b(\lim_{{x \to c}}f(x))\)
  2. \(\lim_{{x \to c}}(f(x) + g(x)) = \lim_{{x \to c}}f(x) + \lim_{{x \to c}}g(x)\)
  3. \(\lim_{{x \to c}}(f(x)g(x)) = (\lim_{{x \to c}}f(x))(\lim_{{x \to c}}g(x))\)
  4. \(\lim_{{x \to c}} \frac{f(x)}{g(x)} = \frac{\lim_{{x \to c}} f(x)}{\lim_{{x \to c}} g(x)}\), provided \(\lim_{{x \to c}} g(x) \neq 0\)
  5. For any constant k, \(\lim_{{x \to c}} k = k\)
  6. \(\lim_{{x \to c}} x = c\)
  7. \(\lim_{{x \to c}} f(g(x)) = f(\lim_ {{x \to c}} g(x))\)

Algebraic Manipulation to Solve for Limits

Sometimes we can use algebra to simplify a limit & arrive at a finite answer.

Example: Solve for \(\lim_{{x \to 2}} \frac{x^2 + x - 6}{x^2 - 4}\)

By using direct substitution, we get,
\(\lim_{{x \to 2}} \frac{x^2 + x - 6}{x^2 - 4} = \frac{2^2 + 2 - 6}{2^2 - 4} = \frac{0}{0} = DNE\)
BUT if we simplify the expression first, we get,
\(\lim_{{x \to 2}} \frac{x^2 + x - 6}{x^2 - 4} \) \(= \lim_{{x \to 2}} \frac{(x + 3)\cancel{(x - 2)}}{(x + 2)\cancel{(x - 2)}} = \lim_{x \to 2} \frac{x + 3}{x + 2} = \frac{2 + 3}{2 + 2} = \frac{5}{4}\)
& viola! the limit does exist!!
