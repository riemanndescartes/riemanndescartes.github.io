SO, What is a limit?

A limit is the value a function approaches as the variable of the function gets closer & closer to a particular value.

Limit Notation:

Note:
Notice how the word "approaches" is used.
This means that f(c) does not have to equal L.
It just has to approach L from both sides.

We write if the value ƒ(x) approaches L as x approaches c.

One-Sided & Two-Sided Limits

limx→2 ƒ(x) is the number that ƒ(x) approaches as x approaches 2 from both sides.

here, limx→2 ƒ(x) = L1
This is a two-sided limit (aka the general limit)

limx→2+ ƒ(x) is the number that ƒ(x) approaches as x approaches 2 only with values greater than 2.

↪ this is called a right-hand limit.

limx→2- ƒ(x) is the number that ƒ(x) approaches as x approaches 2 only with values less than 2.

↪ this is called a left-hand limit.

limx→2+ ƒ(x) = L2
limx→2- ƒ(x) = L3
